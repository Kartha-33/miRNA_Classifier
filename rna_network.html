🧬 RNA Network Analysis

Interactive visualization of sequence relationships and clustering patterns

🎨 Legend

Blue = miRNA sequences
Dark = non-miRNA sequences
Node size = Sequence length

📊 What Are The Clusters?

Central clusters (dense regions) represent:
  • Sequence Families: Groups of RNA sequences with similar k-mer patterns
  • Functional Similarity: Sequences that likely share biological functions
  • Structural Similarity: Similar secondary structure motifs
  • Evolutionary Relationships: May have common evolutionary origin
Why sequences cluster together:
  • High k-mer similarity (3-base patterns)
  • Similar GC content
  • Shared sequence motifs
  • Related biological function

🔍 How To Interpret

The connections (edges) show:
  • Sequences with >30% similarity
  • Thicker clusters = more similarity
  • Isolated nodes = unique sequences
  • Bridge nodes = connect different families

⌨️ Navigation

🖱️ Drag: Pan the view

🔍 Scroll: Zoom in/out

👆 Hover: See sequence details

🎯 Click: Select node

📈 Technical Details

Similarity Calculation:
  • Method: Cosine similarity of k-mer vectors
  • K-mer size: 3 nucleotides
  • Layout: Force-directed (spring)
  • Clustering: Greedy modularity
